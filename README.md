# Yuz - PlayerDataManager
PlayerDataManager is a simple [ProfileStore](https://github.com/MadStudioRoblox/Replica) and [Replica](https://github.com/MadStudioRoblox/ProfileStore) wrapper for player data management, that handles data saving and replication.
